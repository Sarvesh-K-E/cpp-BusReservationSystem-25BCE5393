## 📖 Overview

**Bus Reservation System** is a high-performance, modern bus reservation platform. Combining a sleek, dark-themed visual interface with a heavy-duty **C++ backend engine** compiled to WebAssembly, it provides a seamless and responsive booking experience.

The system features real-time seat synchronization, interactive route maps, and a fully functional embedded terminal for power users, all powered by a robust cloud-synced architecture.

## 🚀 Features

- **⚡ C++ WASM Engine**: Blazing fast logic execution for fare calculation and seat management.
- **🛰️ Live Cloud Sync**: Real-time Firebase integration for instant seat locks and ticket persistence.
- **🗺️ Interactive Map**: Specialized route visualization and stop tracking.
- **📟 Power Terminal**: Command-driven interface for lightning-fast bookings and system diagnostics.
- **🎨 Modern Aesthetics**: Premium dark-mode UI with smooth micro-animations and custom glassmorphism components.
- **🛡️ Admin Suite**: Full-featured administrative panel for route management, bus scheduling, and revenue reporting.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Backend Core**: C++17 (compiled via Emscripten to WASM)
- **Database**: Firebase Realtime Database & Auth
- **3D Environment**: Spline (Interactive Viewport)
- **Styling**: Custom Design System

## 📂 Project Structure

- `/cpp`: Core C++ Reservation engine source.
- `/dist`: Compiled WebAssembly artifacts (`.wasm`, `.js`).
- `app.js`: Main UI logic and synchronization layer.
- `index.html`: Optimized application entry point.
- `styles.css`: Custom premium design tokens.
